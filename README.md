# Cat UI - Um Design System Angular

> Cat UI é um Design System para projetos Angular, onde seu maior objetivo é abstrair ao máximo a criação de sistemas, permitindo mais produtividade.

## Instalação

```bash
npm i @catrx/ui
```

## Compatibilidade

| @catrx/ui | Angular |
| :-------: | :-----: |
|   1.x.x   | 15.0.4  |
|   2.x.x   | 16.0.0  |
|   3.x.x   | 17.0.0  |
